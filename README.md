# Guia para ejecutar el programa:
- Abrir editor de texto e ir a [Archivos/Files]
- Luego seleccionar [Abrir carpeta/Open Folder]
- Seleccionar la carpeta ./PP-Juego/Juego - PP    <---
- Ejecutar y/o compilar el código.


# Jugabilidad Básica:
- (1) El juego comienza con el turno del primer jugador (aún el programa señala los turnos por consola y no en pantalla).
- (2) Seleccionar y pulsar con el mouse el recuadro de [Angulo], digitar ángulo, de preferencia entre 0 y 90 grados para el
primer jugador, para confirmar pulsar la barra espaciadora.
- (3) Luego seleccionar y pulsar con el mouse el recuadro de [Velocidad], digitar un numero, de preferencia mayor que 0,
despues pulsar la barra espaciadora para confirmar y se ejecutara el tiro.
- Para el segundo jugador repetir los pasos (2) y (3) respectivamente, se recomienda un angulo entre 90 y 180 grados y una
velocidad mayor que 0.


*El programa en su version actual permite la jugabilidad, pero hay detalles que se irán añadiendo a futuro para mejorar la
jugabilidad e interaccion con el usuario*
